# ECommerce QA Automation Framework

A Selenium WebDriver Automation Framework developed using Python and PyTest.

## Features

- Selenium WebDriver
- Python Automation
- PyTest Framework
- Page Object Model (POM)
- Explicit Waits
- Data Driven Testing
- End-to-End Test Scenarios

## Automated Scenarios

- User Login Validation
- Invalid Login Validation
- Product Search
- Add Product to Cart
- Checkout Workflow

## Technologies Used

- Python
- Selenium
- PyTest
- JSON Test Data
- ChromeDriver
